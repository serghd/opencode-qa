# AGENTS.md

## Project Overview
Static Q&A markdown files generated by LLM for interview preparation. No code, no build system, no tests.

## Structure
- `nodejs-qa.md` — 46 Node.js interview questions (fundamentals, async, Express, streams, crypto, worker threads, etc.)
- `postgres-qa.md` — 5 PostgreSQL interview questions (basics, MVCC, indexes, CTEs, comparisons)
- `sql-qa.md` — 13 SQL interview questions (joins, grouping, subqueries, indexes, data types, aggregate functions, CTEs, EXISTS vs IN, views)
- `solidity-qa.md` — 20 Solidity smart contract interview questions (basics, visibility, data locations, security, Ether transfers, mappings, libraries, error handling)

## Commands
- None. No `package.json`, linter, formatter, or CI configured.

## Conventions
- Content is plain Markdown. Edit in place.
- New Q&A topics should follow the existing `nodejs-qa.md` format (numbered questions with concise answers).
