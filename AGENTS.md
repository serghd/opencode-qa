# AGENTS.md

## Project Overview
Static Q&A markdown files generated by LLM for interview preparation. No code, no build system, no tests.

## Structure
- `nodejs-qa.md` — 44 Node.js interview questions (fundamentals, async, Express, streams, crypto, worker threads, etc.)
- `postgres-qa.md` — PostgreSQL interview questions
- `solidity-qa.md` — 10 Solidity smart contract interview questions (basics, visibility, data locations, security, Ether transfers)

## Commands
- None. No `package.json`, linter, formatter, or CI configured.

## Conventions
- Content is plain Markdown. Edit in place.
- New Q&A topics should follow the existing `nodejs-qa.md` format (numbered questions with concise answers).
